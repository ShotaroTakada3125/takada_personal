
# 要件定義書：SyncroLife (Ver. 2.0)

## 1. プロジェクト概要
* **名称:** **SyncroLife (シンクロライフ)**
* **コンセプト:** 「考えない、迷わない。」―― 思考の断片から、最適な行動をデザインするAIライフマネージャー。
* **ターゲット:** 自己研鑽（勉強）とプライベートの両立に悩み、タスク管理自体に疲弊しているユーザー。

---

## 2. 機能要件 (Functional Requirements)

### ① 思考の構造化・プロジェクト管理
* **プロジェクト自動生成:** ユーザーの曖昧な思考入力をトリガーに、1つの「プロジェクト」を生成。
* **タスク分解 (WBS):** プロジェクトを達成するために必要な具体的ステップ（タスク）へ分解し、依存関係（順序）を設定。
* **ハイブリッド進捗管理:** * **タスク進捗:** 各タスクの完了状態（未着手・進行中・完了）。
    * **プロジェクト進捗:** タスクの重み付けに基づいた、全体の達成率（%）を算出。

### ② 多次元スコアリング・エンジン
タスクの優先順位を決定するための動的パラメータ群。

| パラメータ | 内容 | 自動設定のロジック |
| :--- | :--- | :--- |
| **期限 (Deadline)** | 完了すべき最終日時 | カレンダー・入力から推定 |
| **労力 (Effort)** | 推定作業時間・精神的負荷 | 過去の類似タスクから算出 |
| **進捗 (Progress)** | 0〜100% | 完了タスク数・入力から算出 |
| **場所 (Context)** | 自宅、オフィス、外出先、移動中 | GPS、Wi-Fi、予定から判定 |
| **気分/活量 (Energy)** | 集中力の高さ、疲労度 | 睡眠・活動データから算出 |
| **重要度 (Value)** | 人生や目標へのインパクト | AIがプロジェクトの質から判定 |
| **依存性 (Dependency)** | 先行タスクの有無 | プロジェクト構造から自動反映 |

### ③ プロアクティブ・スケジューリング（外出提案含む）
* **能動的な外出提案:** * 一日の予定が空いている場合、溜まっている「外出先タスク」を検出し、移動効率や天気を加味した外出プランを提案する。
* **柔軟な定期タスク:** * 「7日に1回」などの周期タスクを、固定枠ではなく「空き時間かつリソースがある時」に動的に差し込む。

### ④ 認知負荷ガード（AIによる「断る」提案）
* **オーバーフロー検知:** 予定とタスクの総負荷が、ユーザーのキャパシティ（睡眠・体調データから算出）を超えた場合に警告。
* **断捨離・延期提案:** * **延期:** 「今週は無理です、来週に回しましょう」
    * **削除:** 「3ヶ月未着手です、本当に必要ですか？」と問いかけ、不要なタスクを整理。

### ⑤ 学習型フィードバック・ループ
* **行動学習:** ユーザーが実際にタスクをこなした時間帯や所要時間を学習し、次回の予測精度を向上。
* **先延ばしパターンの検出:** よく後回しにするタスクを特定し、「タスクの再分割」や「最適なタイミングへの変更」を自動提案。

### ⑥ ライフログ・インテグレーション
* **カレンダー連携:** 空き時間の特定と、予定に付随するTODOの自動生成。
* **生体データ連携:** 睡眠の質や活動量に基づき、その日の「最大実行可能スコア」を決定。
* **環境データ:** 天気・気温に基づき、屋外タスクの実行可否を自動調整。

---

## 3. 非機能要件 (Non-functional Requirements)

* **UX/UI:** * ユーザーに「命令」するのではなく、優秀な秘書のように「提案」するトーン。
    * 複雑なパラメータを意識させない、シンプルで直感的なダッシュボード。
* **プライバシー:** ライフログやカレンダー等の機密情報は、暗号化の上、厳重な権限管理のもとで処理。
* **低遅延:** 思考入力からタスク分解・スケジューリングの提示までを数秒以内に行う。

---

## 4. 拡張アイデア（優先度：低）

* **ゲーミフィケーション:** タスク完了や体調管理の状態を「学習レベル」や「健康スコア」として可視化するダッシュボード機能。

### SyncroLifeの進捗算出ロジック（例）
プロジェクトの全体進捗率 $P$ は、各タスク $i$ の重み（労力） $w_i$ とその達成度 $d_i$ を用いて以下のように定義されます。

$$P = \frac{\sum_{i=1}^{n} (w_i \times d_i)}{\sum_{i=1}^{n} w_i} \times 100$$
