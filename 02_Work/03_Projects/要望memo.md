## プロンプト

勉強や個人的に欲しくて、最強のtodoリストを作成したい。

普通のtodoリストではなく、生活をより豊かに楽になるようなアプリ・サービスです。


普通のtodoリストにある機能ってなんだと思う？

# アイデア

1. ユーザーがざっくり思考を入力すると、タスクに分解する
  - 思考を入力して、その思考の要望及びその過程で必要な複数のタスクに分解する
    - 例：「卒業旅行どうする？」→ チケット比較・パスポート確認・ホテル候補探し
  - 自分でタスク分解するのがめんどくさいので、自動でやってくれるようにしたい
  - 思考の断片 → 要望・自動整理　→ 細かいタスク化

2. タスクの優先度を捉えるものとして要素(優先度パラメータ)を作成する
  - 優先度パラメータの要素としては、[期限,労力,進捗,場所,気分]を考えている。他にあるか？
  - 各要素のパラメータ設定については最終的には自動でやってくれるようにしたい

3. 優先度パラメータの修正/編集
  - 各タスクのパラメータを手動で修正できるようにしたい
  - 過去の行動データから自動最適化
    - 完了時間の傾向や時間帯、進捗の進み方を学習
    - よく先延ばしにするパターンを検出して改善案を自動で出すイメージ

4. ライフログとの連動
  - 体調・負荷・予定を加味してタスク提案を変える
    - カレンダーと連携し、予定を鑑みてタスクに充てることのできる時間を推測
    - カレンダーの予定を取得し、todoを自動生成？
    - 睡眠データと連動できればパフォーマンスによって一日の実行可能タスクの予想をする

5. タスクのプロジェクト化
  - 大きな目標（例：筋トレ習慣化）を登録
  - 自動で小さいタスクへ分解
  - 実行順序をAIが提示（“次はこれやるといいよ”）

6. 定期タスクの柔軟な自動提案
  - 入力例：「部屋の掃除：7日周期」「散髪：月1回」
    - 「毎週日曜」ではなく、期間内ならいつでもOKなタスクとして扱う
  - AIが提案する例：
    - 今週は忙しそう → 後ろにずらす
    - 今日在宅＆時間ありそう → 今日やる？と提案
    - 最近疲れてそう → 明日に回す案
    - 天気が悪い → 買い物系タスクは後ろへ
