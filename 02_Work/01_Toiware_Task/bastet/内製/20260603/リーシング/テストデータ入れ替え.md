- 本番用のキャンペーンの開始期間を期間中にする

```sql
select * from stockholder_bonus_master;

update stockholder_bonus_master set entry_start_datetime = '2026-04-28 13:00:00' where id in (1,2);
```

- 現在の期間中のキャンペーン
  - 名称を期間後にする
    ```sql
    UPDATE stockholder_bonus_master 
    SET title = 'テスト用SBIインシュアランスグループ株主優待（期間後）' 
    WHERE id = 7;

    UPDATE stockholder_bonus_master 
    SET title = 'テスト用SBIアルヒ株主優待（期間後）' 
    WHERE id = 9;
    ```
  - 開始期間をずらす
    ```sql
    update stockholder_bonus_master set entry_start_datetime = '2026-07-01 13:00:00' where id in (7,9);
    ```
- テストデータの紐付きを変更する
  - テスト中100件、期間前、期間後20件
  ```sql
  UPDATE stockholder_bonus_entry
  SET stockholder_bonus_master_id = CASE 
      WHEN stockholder_bonus_master_id = 1 THEN 9
      WHEN stockholder_bonus_master_id = 9 THEN 1
      ELSE stockholder_bonus_master_id -- 基本は上のIN句で絞るので通りませんが念のため
  END,
  update_datetime = CURRENT_TIMESTAMP, -- 更新日時も記録する場合
  update_user = 'manual_fix'           -- 誰がやったか残す場合
  WHERE stockholder_bonus_master_id IN (1, 9);
  ```
