# 口座開設フロント

- ステーキング報酬受取設定
- テスト対象：個人本登録, 法人口座開設完了, 法人本登録の3パターンで確認
- テスト観点
  - XTZ非表示確認
    - マイページログイン時、ステーキング報酬受取設定の銘柄一覧に XTZ が存在しないこと
    - マイページがfigmaのデザイン通りになっていること
- 正常性確認：個人本登録、法人本登録の2パターンで確認
  - 初期表示
    - 現在の設定が全て　受取 になっていること
    - チェックボックスが全てオフになっていること
  - チェックボックス切り替え
    - 切り替えが問題ないこと
    - 画面の表示が崩れないこと
  - 設定変更 (受取 → 拒否)
    - 設定変更した銘柄の設定が 拒否 になること
    - チェックボックスがオンになっていること
    - 画面遷移やログアウトで状態が変化しないこと
  - 設定変更 (拒否 → 受取)
    - 設定変更した銘柄の設定が 受取 になること
    - チェックボックスが オフ になっていること
    - 画面遷移やログアウトで状態が変化しないこと
  - 設定未反映 (受取 ↔︎ 拒否)：チェックボックスだけ変更し、「変更する」ボタンを押下しない状態
    - 受取設定、チェックボックスの状態がリセットされていること

## 正常性確認：リグレッションテストで担保のためITbでは打鍵不要

- ステーキング報酬額照会検索画面
  - 初期表示がfigmaと一致すること
  - 報酬総額csvファイルアップロード
    - 下記条件でファイルアップロードがエラーになること
      - 不正なファイル形式
      - ヘッダーなし
      - ヘッダーのみ
      - 存在しない銘柄
      - ステーキング対象外の銘柄
      - 報酬総額カラムが数字以外
      - 顧客報酬割合カラムが数字以外
      - 顧客報酬割合カラムが0~100の整数以外
    - 任意の2銘柄を同時に新規登録/更新それぞれできること
  - 先月分のデータ検索
    - 検索条件なしでエラー
    - XTZ 以外の銘柄で検索すると 合計金額 と　通常口座ステーキング報酬 に該当のデータが表示されること
    - 「追加口座の登録」を押下すると ステーキング報酬追加口座照会検索画 に遷移すること
  - 通常口座ステーキング報酬 ファイル出力
    - 最終付与額一覧CSV出力
      - 銘柄指定なしでエラーになること
      - XTZ が指定できないので、XTZのファイル出力ができないこと
    - 入庫用リスト出力
      - 銘柄指定なしでエラーになること
      - XTZ が指定できないので、XTZのファイル出力ができないこと
    - 全銘柄入庫用リスト出力
      - モーダルをキャンセルで閉じることができること
      - 出力結果に XTZ のCSVファイルが存在しないこと
- ステーキング報酬追加口座照会検索画面
  - 初期表示がfigmaと一致すること
  - 検索
    - 検索条件なしでエラー
    - 任意の銘柄で検索すると対象のデータが表示されること
    - 人氏の銘柄と口座番号で検索すると対象のデータが表示されること
  - 登録
    - 条件なしで登録ボタンを押下するとエラーになること
    - 任意の銘柄で口座名と月間保有数量合計を空白の状態で登録ボタンを押下するとエラーになること
    - 任意の銘柄で月間保有数量合計を数字以外の状態で登録ボタンを押下するとエラーになること
    - 任意の銘柄で口座名と月間保有数量合計を正しい状態で入力し登録ボタンを押下すると画面上に入力した検索条件の口座が登録されること
    - 登録したデータが存在する状態で、再度検索すると画面上に登録したデータの口座が表示されること
  - 追加口座の削除
    - モーダルをキャンセルで閉じることができること
    - 任意の銘柄の口座を1件, 複数件削除できること
- ステーキング手数料キャッシュバック額照会検索画面
  - 初期表示がfigmaと一致すること
  - キャッシュバック対象口座csvファイル
    - 下記条件でファイルアップロードがエラーになること
      - 不正なファイル形式
      - ヘッダーなし
      - ヘッダーのみ
      - 存在しない銘柄
      - ステーキング対象外の銘柄
      - 口座番号カラムが数字以外
      - 口座番号カラムが8桁以外
    - 任意の2銘柄が同時に新規登録/更新できること
  - キャッシュバック割合CSV入力
    - 下記条件でファイルアップロードがエラーになること
      - 不正なファイル形式
      - ヘッダーなし
      - ヘッダーのみ
      - 存在しない銘柄
      - ステーキング対象外の銘柄
      - キャッシュバック割合カラムが数字以外
      - キャッシュバック割合カラムが0~100の整数以外
    - 任意の2銘柄が同時に新規登録/更新できること
  - 検索
    - 銘柄未選択で検索するとエラーかつ、「キャッシュバック額一覧CSV出力」ボタンと「入庫用リスト出力」ボタンが非活性であること
    - 銘柄全てで検索すると、データが表示され、「キャッシュバック額一覧CSV出力」ボタンと「入庫用リスト出力」ボタンが活性であること
    - 銘柄全て, 登録した口座番号 で検索すると、データが表示され、「キャッシュバック額一覧CSV出力」ボタンと「入庫用リスト出力」ボタンが活性であること
    - 任意の銘柄で検索すると、データが表示され、「キャッシュバック額一覧CSV出力」ボタンと「入庫用リスト出力」ボタンが活性であること
    - 任意の銘柄, 登録した口座番号 で検索すると、データが表示され、「キャッシュバック額一覧CSV出力」ボタンと「入庫用リスト出力」ボタンが活性であること
  - キャッシュバック額一覧CSV出力
    - 銘柄全てで検索し、「キャッシュバック額一覧CSV出力」ボタンを押下すると、全ての銘柄が含まれたCSVファイルがダウンロードできること
    - 任意の銘柄で検索し、「キャッシュバック額一覧CSV出力」ボタンを押下すると、検索条件の銘柄が含まれたCSVファイルがダウンロードできること
  - 入庫用リスト出力
    - 銘柄全てで検索し、「入庫用リスト出力」ボタンを押下すると、全ての銘柄が含まれたCSVファイルがダウンロードできること
    - 任意の銘柄で検索し、「入庫用リスト出力」ボタンを押下すると、検索条件の銘柄が含まれたCSVファイルがダウンロードできること
